## 💻 Getting Started

### Clone the repository

```bash
git clone https://github.com/veenasingh-hq/react-portfolio-practice.git
```

### Navigate to the project

```bash
cd react-portfolio-practice
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```
